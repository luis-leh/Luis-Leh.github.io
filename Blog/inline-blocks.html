Inline vs. Inline-Blocks

What does it mean?

Let's find out below

Inline

  • Inline elements are displayed on the same line
  • Setting height and width properties for inline elements will not have an effect
  • Inline elements will only take as much width as its content

Inline-Block

  • Inline-block elements are displayed on the same line but within a box or contatiner
  • Setting height and width properties are possible

Block

  • Block elements are displayed on a new line
  • Block elements take up the full available width by default
  • Setting height and width properties are possible
